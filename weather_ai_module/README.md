Weather AI Module

## Описание
Модуль искусственного интеллекта для анализа погодных условий и выдачи рекомендаций по активности.

## Состав модуля
- `WeatherAI.java` — основной класс с логикой анализа погоды для 5 активностей (рыбалка, бег, пикник, велосипед, поход).
- `WeatherLogger.java` — класс для логирования работы системы.
- `HistoryManager.java` — класс для сохранения истории запросов в CSV-файл.
- `NotificationClient.java` — класс WebSocket-клиента для отправки уведомлений (требует настройки URL сервера).
- `WeatherData.java` — класс модели данных погоды.
- `Recommendation.java` — класс модели рекомендации.
- `Main.java` — демонстрационный класс для тестирования модуля.
